# 🌱 습관 정원 (Habit Garden)

매일 물을 주며 키우는 나의 습관들을 **정원의 식물**로 번역하는 서비스.
좋은 습관은 자라는 화초, 나쁜 습관은 뽑아도 또 나는 잡초, 미루기는 시들어가는 화분입니다.

## 무엇을 하나요 (데일리 습관 트래커)
1. 처음 한 번: 매일 키울 습관 3~5개 + ‘정체성’ 선택
2. **매일 접속 → 습관별 ‘물주기’ 체크인**
3. 날짜 기준으로 **연속일(streak)** 이 실제로 쌓이고, 거르면 다음 날 목마름 → 시듦
4. **21일 잔디 캘린더**(GitHub 스타일)로 매일의 물주기를 시각화
5. 가장 뒤처진 습관엔 정원사 ‘흙손’의 **오늘의 처방**(신호·2분 규칙·보상·환경)

기록은 브라우저 `localStorage`에 저장되어, 다시 방문하면 이어집니다. (서버·AI 없이 동작)

## 행동과학 근거
- **습관 쌓기(habit stacking)** — 기존 행동 ‘직후’에 새 습관을 앵커링
- **2분 규칙** — 문턱을 최소 단위로 낮춰 시작 저항 제거
- **신호–루틴–보상 루프** — 즉각 보상으로 행동을 강화
- **정체성 기반 습관** — ‘무엇을 한다’가 아니라 ‘어떤 사람이 된다’로 프레이밍
- **자동화 곡선** — 습관 자동화 중앙값 ≈ 66일(Lally et al., 2010)을 진행률로 시각화

## 실행
```bash
npm install
npm run dev
# http://localhost:3000
```

## 스택
Next.js 14 (App Router) · TypeScript · Tailwind CSS

## 구조
```
app/
  layout.tsx      # 메타/레이아웃
  page.tsx        # 서비스 홈 = 정원
  globals.css
components/
  HabitGarden.tsx # 온보딩 → 데일리 물주기 → 잔디 캘린더 (클라이언트)
lib/
  garden.ts       # 습관 데이터 + localStorage 저장/로드 + 연속일·상태 계산
```

## 확장 아이디어
- 데일리 체크인 + 연속일 저장(로컬/서버)로 실제 습관 트래커화
